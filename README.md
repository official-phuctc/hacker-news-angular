# README

This repository create front server to show Hacker News by talking data from rails backend server

# SET UP AND RUN

## On development
## Install packages
```npm install```

## Config Project

### Config environment
Config enviroment in environments/environment.ts

### Run server
```ng serve```

Go to [localhost:4200](localhost:4200)
